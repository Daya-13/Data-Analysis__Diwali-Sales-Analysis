# Data Analysis-Diwali Sales Analysis
This project analyzes Diwali sales data to understand customer purchasing behavior and sales performance during the festive season. The analysis focuses on identifying key customer segments, high-performing product categories, and revenue trends across demographics.
Using Python (Pandas, NumPy, Matplotlib/Seaborn), the dataset was cleaned, processed, and analyzed to derive meaningful business insights. The results help businesses optimize marketing strategies, inventory planning, and customer targeting during high-demand festive periods.

# Key highlights:
Data cleaning and preprocessing
Exploratory Data Analysis (EDA)
Customer segmentation based on demographics
Sales and revenue trend analysis
Actionable business insights

# Tools & Technologies:
Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook
